# Student Performance Prediction

This is a simple machine learning project where I tried to predict exam scores based on study hours.

## What I did
I created a small dataset showing how study time might affect exam results, and trained a Linear Regression model to learn this relationship.

## Features
- Predicts exam scores based on study hours
- Uses a simple regression model
- Calculates error using MAE
- Shows a basic graph of the results

## Technologies
- Python
- NumPy
- Scikit-learn
- Matplotlib

## How to run
pip install numpy scikit-learn matplotlib  
python3 grade-prediction.py
